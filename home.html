Course Description.

The course will be split into three main sections.

  1. The first section will cover the basic principles of web development. From client server architecture, HTTP protocol as well as the basics of web development (HTML & CSS).
  2. The second part will cover Java-script and from end development
Tools

Any IDE or code editor of your choice (Visual Studio, Sublime Text, Netbeans, Notepad++ etc). You will need MySQL (probably already installed from last term) as well as NodeJS installed.

We will use GitHub for our code management and Heroku for hosting our projects.

You will also need to install Postman for API testing (Links to an external site.))

 Assignments

This course will have 5 assignments, and a final project. (A Summative). The Summative will be a group project. You will be required to form groups of two. There will be two individual take home assignments and two group take home assignments, and an In-Class Quiz.

The final project will build on the work we did for the Database Management Systems Class, and we will build a web application for manipulating that database.

 Grade Weighting
  • Attendance/Participation - 10%
  • Quiz - 10%
  • Take Home Assignments - 40%
  • Final Project - 40%

You should treat me as a client. I will give you a set of requirements. Requirements can be interpreted differently, this is unavoidable. You have to meet my interpretation of the requirements to get maximum credit. If you have any doubt about what a requirement means, you should ask me to clarify.
